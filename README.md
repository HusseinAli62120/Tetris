# 🎮 Flutter Tetris


The classic, block stacking game, built purely using flutter.

👉 Download here: https://drive.google.com/file/d/1DXKr65ai2FcYnTJMAFwpXtQR003zMiS4/view?usp=drive_link

### 🚀 Features:

- 🧩 Classic Tetris gameplay
- 🎯 Score & Time tracking system
- 🎮 Intuitive touch controls

### 💻 Getting started:

1. Clone the repository.
2. Navigate to the project, and run: `flutter pub get` in the termial.
3. After the packages are installed, run: `flutter run` to start the project.

📌 Or you can simply download the app from the link at the start.

### 📦 Packages used:

The following packages were used in the creation of this app:

- **flutter_animate:** Used for basic animations.
- flutter_launcher_icons: Used to generate the app icon.
- flutter_native_splash: Used to generate the app’s splash screen.
- hive & hive_flutter: Used to store high scores.
